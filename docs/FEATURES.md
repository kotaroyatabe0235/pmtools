# PM Tools - Features

ブラウザで動作する Microsoft Project 的なプロジェクト管理ツールの機能一覧。

## Feature Categories

### 1. プロジェクト管理

#### 基本機能
- [x] プロジェクト作成（フォーム入力）
- [x] プロジェクト編集
- [x] プロジェクト削除
- [x] プロジェクト詳細表示
- [x] プロジェクト一覧表示（検索・ソート）
- [x] プロジェクト複製
- [x] プロジェクトアーカイブ

#### 詳細機能
- [ ] プロジェクトテンプレート
- [ ] プロジェクト比較
- [ ] プロジェクト履歴・バージョン管理
- [ ] プロジェクト共有

#### プロジェクト情報
- プロジェクト名
- プロジェクト記述
- 開始日（必須）
- 終了日（可変）
- 担当者（所有権）
